# Sequence Method Info

This fold contains 3 K562 multiomics datasets generated in Xu et al, each generated with different sequencing method. 

1. 10X_multiomics: Generated by 10x multiomics kit and aligned via 10x cloud CellRanger ARC 2.0.2. 

2. ISSAAC-seq: Generated by Xu et al's method in the orginal paper, aligned with customized code, STAR, and Chromap. 

3. Share-seq: Generated by Xu et al as comparison to ISSAAC-seq, aligned with STAR.
